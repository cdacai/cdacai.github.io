wordPress快速建站日志——正式开通

Posted on 2019-01-22 Edited on 2019-12-31 In web前端 Valine:

昨天备案成功，出了个小插曲——

网站首页可以访问，但是所有图片没有显示，点击文章和链接会跳转到让输入ftp服务密码的临时域名，这个差错一度让我觉得是阿里的主机问题或者我浏览器有问题。

解决方法：

还是阿里的工单那里的帮助文章说的，是wordPress的建站时候的网址的设置问题，改过来，终于好了（备注，这个站点地址输入框上面那个wordPress访问的网址也要改过来，不然登陆界面也是跳转到未备案域名，好坑）。

开通的正式思考：

1、教训：

这个网站的建设基本上是失败的。

虚拟主机的数据库、服务程序都已经内置好了，甚至连后端语言php都限定死了，不允许有任何种类更改。

2、收获

建站的完整流程知道了，也算没白搭建，为下一次做准备

3、打算

申请云服务器，这才是真正想要的，从头到尾搞起。

搞好再把www.caixiaohu.com这个域名解析过去。

New Start——云服务器搭建日志-1-连接并安装java

无意中看到阿里云服务器有免费试用1个月，第一年续费仅99元。

配置是1g1核，

40g系统盘（系统大概占用了3g，剩下35g左右的空间，也可以存自己的东西，所以个人建站来说空间足够用）

专有网络（就是说没有专用的ip，是从外网租借转发而来的，效果一样。）

不用傻逼，果断开搞，申请到手。

下面是两种连接和安装java软件的过程。

1、ssh连接（输入命令用）

首先要注意，阿里云已经默认内置了安全规则，允许端口22远程ssh连接，不是阿里云的要注意去主机控制台里给远程ssh连接加权限。

其次要去重置下实例服务器的密码，也方便一会的ssh连接输入密码。

开始：

阿里自有内置自带网页版远程连接命令行，但上下翻动时分不方便，所以用mac机自带的终端。

mac机打开终端

输入： ssh 你的服务器实例登陆用户名@你的公网ip

回车后输入密码，就会出现阿里云的欢迎界面，如下图：

2、软件连接（上传文件用）

下载FileZilla，安装

输入连接1里面的ip，用户名，密码，和端口号22，快速连接，

出现成功界面，服务器上的文件目录就出来啦，如下图

3、安装java

我的云服务器选择的系统是ubuntu系统（据说比较人性化、易用，常见的centos更适合企业用户）

所以下面的安装是按照ubuntu系统来的。

安装步骤如下：

注意：阿里云服务器第二步安装会报错（add-apt-repository: command not found），是因为缺少文件，安装即可。

解决方法：输入命令 sudo apt-get install software-properties-common python-software-properties

一.安装jdk

 首先通过xshell连接到服务器

 然后首先输入命令:sudo su来获取权限,因为ubuntu系统对权限的要求比较高,不能以root用户登录

1.默认jdk安装

  sudo apt-get update

  sudo apt-get install default-jre

 2.Oracle JDK 安装（推荐）

   设置 PPA

  sudo add-apt-repository ppa:webupd8team/java

  sudo apt-get update

   JDK6 ：

  sudo apt-get install oracle-java6-installer

 JDK 7：

 sudo apt-get install oracle-java7-installer

 JDK 8

  sudo apt-get install oracle-java8-installer

安装完毕输入java -version,出现了jdk版本说明已经安装好了

输入：java -verbose可以查看java安装路径

3.设置jdk环境变量

使用vi打开配置文件：sudo vi ~/.bashrc 然后按i开始添加

在末尾添加：

export JAVA_HOME=/usr/lib/jvm/java-8-oracle

export JRE_HOME=${JAVA_HOME}/jre

export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib

export PATH=${JAVA_HOME}/bin:$PATH

再按Esc退出,然后输入:wq进行保存，最后使用source ~/.bashrc使刚刚的配置生效

这里注意前后不要有空格,还有如果jdk安装路径不一样,可以用java -verbose查看路径后自己修改
